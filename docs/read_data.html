read_data module¶

read_data.import_data(directory)¶

This function reads the csv files into a panda dataframe.

The function loads all data from the csv file to a pandas dataframe, appendings two headers: Time and Voltage for easier access.

Parameters:directory (string) – File path to the csv file in string format.
Returns:A pandas dataframe array with all data in the csv file.
Return type:dataframe
